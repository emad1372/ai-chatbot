# ai-chatbot
Intelligent chatbot using Python and NLP techniques
This project is an intelligent chatbot developed as part of a university Digital Technologies project.

The chatbot combines natural language processing, weather data integration, and sensor-based temperature monitoring.

Features

Natural language question answering

FAQ and knowledge base system

Weather data retrieval via OpenWeather API

Location information services

Compound question processing

Quiz functionality

Logging and unit testing

Command-line interface support

Technologies

Python

NLP techniques

REST APIs

CSV data handling

Sensor integration (Sense HAT)
